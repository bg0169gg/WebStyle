Using CSS and JavaScript

This guide is about using and applying styles and functions to your static or dynamic webpage.

How can we attach CSS to elements?

Method 1. - Inline CSS

Inline CSS is applied directly to HTML elements using the style attribute.
Code:

<p style="font-size: 18px; color: #ff00ff; letter-spacing: 2px;">Hello world!</p>
Output:

Hello world!

Method 2. - Internal CSS

Internal CSS is defined within the 'style' tag in the head section of an HTML document. This method is useful for styling a single document.
Code:

<head> <style> #demo-p { font-size: 18px; color: #ff00ff; letter-spacing: 2px; } </style> </head> <p id="demo-p">Hello world!</p>
Output:

Hello world!

Method 3. - External CSS

External CSS is linked to an HTML document using the link tag. This method is ideal for applying styles across multiple pages.
Code:

<head> <link rel="stylesheet" href="style.css"> </head> <p id="demo-p">Hello world!</p> <!--Inside style.css--> #demo-p { font-size: 18px; color: #ff00ff; letter-spacing: 2px; }
Output:

Hello world!

How can we attach JavaScript listeners to elements?

Method 1. - Inline functions

Inline JavaScript is added directly to HTML elements using event attributes like onclick, onload, onmouseout, etc.
Code:

<button style="background-color: aqua; font-size: 16px;" onclick="alert('Button pressed!')">Press me!</button>
Output:

Method 2. - Internal functions

Internal JavaScript is placed within the 'script' tag in the head or at the end of the body section.
Code:

<head> <script> function showMessage() { alert('Button pressed!'); } </script> </head> <button style="background-color: aqua; font-size: 16px;" onclick="showMessage()">Press me!</button>
Output:

Method 3. - External functions

External JavaScript is linked to an HTML document using the 'script' tag with the src attribute. This is useful for reusing scripts across multiple pages.
You can use the attribute 'defer' at the end to specify loading the file after all DOM elements are loaded to avoid the script running before page load and prevent syntax errors.
Code:

<head> <script src="functions.js" defer></script> </head> <button style="background-color: aqua; font-size: 16px;" onclick="showMessage()">Press me!</button> <!--Inside functions.js--> function showMessage() { alert('Button pressed!'); }
